JAVA环境搭建

Posted on 2018-04-19

一、环境搭建

  • JRE: java Runtime environment （java 运行环境）

jvm: java virtual machine (java 虚拟机) java 程序就是在 jvm 中运行的， 虚拟机的启动， 只要环境中装有 JRE 环境的话， 运行java代码虚拟机自动启动

java 核心的类库： jre提供的让程序能运行的 java 类

  • JDK ：java development kit (java 开发工具)。

我们只要安装jdk就可以了。
Mac 安装JDK: 访问Oracle官网， 浏览到首页底部菜单， 然后按下如图示操作:

  1. 点击 JDK DOWNLOAD按钮

  2. 选择 Accept Lisence Agreement同意协议:

4.点击 osx_x64_bin.dmg 后面的下载链接：

  1. 打开Finder，可以在下图所示的路径中找到安装好的jdk 1.8.0_40.jdk

5.1 如果找不到资源文件中的java目录， 可以使用一下命令进行查找
$ ls /Library/Java/JavaVirtualMachines/

  1. 其中Contents下的Home文件夹， 是该JDK的根目录。

6.1 使用终端命令查找Home文件夹。

其中:

bin 目录下存放JDK用于开发的一些终端命令巩固， 常见的工具如:
javac的作用是将java源文件编译为class文件（即自解码文件）；
java命令的作用是允许class文件。

db目录下是java开发的一个开源的关系型数据库；

include目录下是一些C语言的头文件。

jre目录下JDK所依赖的java运行时。

lib目录下存放JDK开发工具所依赖的一些库文件；

man目录下存放JDK开发工具的说明文档。

然后输入 java -version, 如果看到jdk版本为1.8说明配置已经生效。

二、 配置环境变量
在英文输入法的状态下，按键盘Ctrl + 空格组合键，调出Spotlight搜索，在这里可以快速启动终端，输入ter,然后回车，即可打开终端：

2.如果你是第一次配置环境变量，可以使用touch .bash_profile 创建一个.bash_profile的隐藏配置文件(如果你是为编辑已存在的配置文件，则使用open -e .bash_profile命令)：

3.输入open -e .bash_profile命令：

4.输入如下配置：

JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk-10.0.1.jdk/Contents/Home
PATH=$JAVA_HOME/bin:$PATH:.
CLASSPATH=$JAVA_HOME/lib/tools.jar:$JAVA_HOME/lib/dt.jar:.
export JAVA_HOME
export PATH
export CLASSPATH

5.使用 source .bash_profile 使配置生效

6.输入 echo $JAVA_HOME 显示刚才配置的路径

  3. 名词解释:
  • 1.Mach-o格式， 我们的程序想要跑起来， 那么程序的可执行文件就需要被操作系统所理解，Mac OS/iOS系统中Mach-o就是可执行文件；Mach-o格式文件有: Executable(可执行文件)、Dynamic Library(动态库)、Bundle(文件)、Static Library(静态库)、Relocatable Object File(可重定向对象文件)都是Mach-o格式文件
    Mach-o可执行文件包含3个段 :

    __TEXT(代码段)： 只读的，包括函数和只读的串
    __DATA: 读写, 包括可读写的全局变量
    __LINKEDIT： 包含了变量的源数据，比如位移、偏移量、包括签名的信息等；

    1. dyld(dynamic loader)动态加载器: 加载一个进程所需要的image
